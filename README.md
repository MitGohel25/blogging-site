# 📝 Blogging Site

A full-stack blogging website built with **Node.js, Express.js, MongoDB, EJS, and JWT Authentication**.

## 🚀 Features
- User Sign Up / Sign In
- JWT Authentication
- Create and View Blogs
- Upload Blog Images
- Add Comments
- Protected Routes
- MongoDB Database

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- JWT
- Multer
